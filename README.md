# 🌍 2026 世界杯竞彩赔率一键拉取

> 世界杯期间，每天跑一行命令，中国体育彩票官方赔率秒级落地到本地。

## 🎯 它能做什么

```
python odds_puller.py

🌍 世界杯 (28 场)  → 赔率全量展开：胜平负 + 让球 + 总进球 + 半全场
📁 其他联赛 (6 场)  → 折叠为一行，不碍眼
```

脚本命中中国体育彩票竞彩网公开 API，每次运行拉取 **全部在售场次**，自动补充半全场和总进球详细赔率，同时产出 JSON 和可读文本两个文件。

---

## 🚀 快速开始

```bash
# 1. 确保 Python >= 3.9
python --version

# 2. 运行（零依赖，只用标准库）
python odds_puller.py
```

输出：

| 文件 | 用途 |
|------|------|
| `latest.json` | 完整 JSON，28+ 场比赛的全部赔率字段 |
| `latest.txt` | 可读文本，世界杯全量展开，其他联赛折叠，方便粘贴给 Claude / ChatGPT 分析 |

## 🧠 搭配 AI 使用

把 `latest.txt` 直接粘贴给 Claude 或 ChatGPT，配合这样的 Prompt：

> 以下是今天的竞彩足球赔率。帮我分析这四场比赛，设计两个 4串4 方案——一个走球数，一个走半全场。两个方案互相对冲，总成本不超过 100 元。

AI 会给出完整的注单计算和逻辑推演。
